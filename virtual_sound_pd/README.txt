Virtual Sound Library for Pure Data - v. 0.4 (2016, October)

INSTALLATION

Put the library folder into the Pd search path


SYSTEM REQUIREMENTS

Pure Data version >= 0.45 on GNU/Linux, Mac OSX, Windows, Raspberry, etc…
